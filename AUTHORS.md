## How to Become a Contributor

We encourage contributions of all types:

- **Publishing provisioners**: the main event — see
  [CONTRIBUTING.md](CONTRIBUTING.md) for the one-line admission flow
- **Code contributions**: fixes and improvements to the scripts, schemas, and
  workflows
- **Documentation**: onboarding guides, examples, clarifications
- **Testing**: bug reports against the data job, validation action, and
  published catalog

## Recognition Policy

Contributors are added to this file when they:

- Submit accepted pull requests with code changes
- Contribute significant documentation improvements
- Provide ongoing community support and assistance
